# Financial Fraud Detection Using Explainable AI and Federated Learning

## Overview

This project aims to detect fraudulent financial transactions using Machine Learning techniques while ensuring transparency and privacy. Explainable AI (SHAP) is used to interpret model predictions, and Federated Learning concepts are incorporated to support privacy-preserving learning.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* CatBoost
* SHAP
* Jupyter Notebook

## Features

* Fraud Detection using Machine Learning
* Explainable AI with SHAP
* Privacy-Preserving Learning using Federated Learning concepts
* Model Evaluation using Accuracy, Precision, Recall, F1-Score, and ROC-AUC

## Project Files

* model.ipynb – Model training and evaluation
* code.ipynb – Data preprocessing and implementation
* test.ipynb – Testing and prediction
