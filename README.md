# Financial Fraud Detection Using Explainable AI and Federated Learning

## Overview

This project focuses on detecting fraudulent financial transactions using advanced Machine Learning techniques while ensuring transparency and privacy. The system leverages Explainable AI (SHAP) to provide interpretable predictions and incorporates Federated Learning concepts to support privacy-preserving model training. The objective is to improve fraud detection accuracy while maintaining trust and security in financial systems.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* XGBoost
* CatBoost
* SHAP (Explainable AI)
* Jupyter Notebook

## Features

* Fraud Detection using Machine Learning models
* Random Forest, XGBoost, and CatBoost implementation
* Explainable AI using SHAP for model interpretability
* Privacy-Preserving Learning using Federated Learning concepts
* Data Preprocessing and Feature Engineering
* Hyperparameter Tuning for model optimization
* Fraud Prediction on financial transaction data
* Performance Evaluation using Accuracy, Precision, Recall, F1-Score, and ROC-AUC
* Transparent prediction analysis through feature importance visualization
* Scalable architecture suitable for financial fraud detection systems
